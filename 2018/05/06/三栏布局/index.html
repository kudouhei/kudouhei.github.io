CSS三栏布局

发表于 2018-05-06 
<section style="margin-bottom: 150px;">
  <!-- 浮动定位 缺点：需要清除浮动；center盒子必须放在最后，中间层后渲染 优点：兼容性好； -->
  <!-- 中间div为何要放在最后？ -->
  <!-- 
    假如某个div元素A是浮动的，如果A元素上一个元素也是浮动的，那么A元素会跟随
    在上一个元素的后边（如果一行放不下这两个元素，那么A元素会被挤到下一行）；
    如果A元素上一个元素是标准流中的元素，那么A的相对垂直位置不会改变，也就是
    说A的顶部总是和上一个元素的底部对齐。
  -->
  <style>
    .left {
      border: 1px solid red;
      float: left;
      width: 200px;
    }
    .center {
      border: 1px solid blue;
      margin: 0 200px;
    }
    .right {
      border: 1px solid yellow;
      float: right;
      width: 200px;
    }
    .box {
      overflow: hidden;
      border: 1px solid gray;
    }
   .sub {
      padding-bottom: 2000px;
      margin-bottom: -2000px;
    }
  </style>
  <div class="box">
    <div class="left sub">left
      <p>left1</p>
      <p>left1</p>
    </div>
    <div class="right sub">right</div>
    <div class="center sub">float布局</div>
  </div>
</section>

<section style="margin-bottom: 150px;">
  <!-- 绝对定位会让整个布局脱离文档流 -->
  <style>
    .wrap {
      position: relative;
    }
    .left1 {
      position: absolute;
      top: 0;
      left: 0;
      border: 1px solid red;
      width: 200px;
    }
    .right1 {
      position: absolute;
      top: 0;
      right: 0;
      border: 1px solid blue;
      width: 200px;
    }
    .center1 {
      margin: 0 200px;
      border: 1px solid gray;
    }
  </style>
  <div class="wrap">
    <div class="left1">left
        <p>left1</p>
    </div>
    <div class="center1">position：absolute布局</div>
    <div class="right1">right</div>
  </div>
</section>

 <!-- flex布局，默认等高 -->
<section style="margin-bottom: 150px;">
  <!-- 兼容性问题；比较主流的移动端布局方式 -->
  <style>
    .wrap3 {
      display: flex;
      width: 100%;
    }
    .left3 {
      width: 200px;
      border: 1px solid blue;
    }
    .right3 {
      width: 200px;
      border: 1px solid red;
    }
    .center3 {
      flex: 1;
      border: 1px solid yellow;
    }
  </style>
  <div class="wrap3">
    <div class="left3">left
        <p>left1</p>
    </div>
    <div class="center3">flex布局</div>
    <div class="right3">right3</div>
  </div>
</section>
