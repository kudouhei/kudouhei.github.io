JavaScript事件

事件级别

  • DOM0：element.onclick = function(){}
  • DOM2: element.addEventListener(‘click’,function(){}, false)
  • DOM3:element.addEventListener(‘keyup’,function(){},false)

事件流(事件三个阶段)
事件流三个阶段，首先捕获阶段，事件通过捕获，到达目标阶段，从目标元素在上传到window对象，为冒泡阶段

描述DOM事件捕获的具体流程
window -> document -> html标签(document.documentElement) -> body ->…->目标元素

Event对象

  • event.preventDefault()阻止默认行为
  • event.stopPropagation()阻止冒泡
  • event.stopImmediatePropagation()注册两个事件，优先级执行
  • event.currentTarget当前绑定事件的对象
  • event.target：触发事件的某个具体的对象，只会出现在事件机制的目标阶段

先捕获，再冒泡
在target注册的监听器，不分捕获和冒泡

事件委托
事件委托是利用事件的冒泡原理来实现的，何为事件冒泡呢？就是事件从最深的节点开始，然后逐步向上传播事件

function delegateEvent(ele, selector, type, fn){
    if(ele.addEventListener){
        ele.addEventListener(type, eventfn, false);
    }else if(ele.attachEvent){
        ele.attachEvent("on"+type, eventfn);
    }else{
        ele['on'+type] = eventfn;
    }
    function eventfn(e){
        var e = e||window.e;
        var target = e.target || e.srcElement;
        if(target.matches(selector)){
            if(fn){
                fn.call(target, e);
            }
        }
    }
}
