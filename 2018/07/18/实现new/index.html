js模拟实现new

new的结果是一个新对象。实例的proto属性会指向构造函数的prototype，也正因为此，实例可以访问原型上的属性。

步骤：

  1. 创建一个新对象
  2. 取出第一个参数，就是要传入的构造函数，因为shift会修改原数组，所有arguments会被去除第一个参数
  3. 将obj的原型指向构造函数，这样obj就可以访问到构造函数原型中的属性。
  4. 使用apply，改变构造函数this的指向到新建的对象，这样obj就可以访问到构造函数中的属性

  5. 返回obj

    function objectFactory(){
    var obj = new Object();
    Constructor = [].shift.call(arguments);
    obj.__proto__ = Constructor.prototype;
    Constructor.apply(obj, arguments);
    return obj;
}

测试代码： 

function Person(name, age){
    this.name = name;
    this.age = age;
}
Person.prototype.strength = 60;
var person = objectFactory(Person, 'Kevin', '18');
console.log(person.name) // Kevin
console.log(person.strength) //60
